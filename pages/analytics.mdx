# Analytics

It's possible to add analytics to your Inertia.js app by declaring it in the Vue.js mounted() callback of your layout file.

<TabbedCodeExamples
  examples={[
    {
      name: 'Vue.js',
      language: 'js',
      code: dedent`
        <template>
        // your layout
        </template>
        <script>\n
        export default {
          props: {
            url: String,
          },
          mounted() {
            // for example, Google Analytics via Google Tag Manager
            var dataLayer = window.dataLayer || [];
            dataLayer.push({
                'event': 'pageView',
                'virtualUrl': this.$inertia.page.url;,
            });
          }
        }
        </script>
      `,
    },
  ]}
/>
